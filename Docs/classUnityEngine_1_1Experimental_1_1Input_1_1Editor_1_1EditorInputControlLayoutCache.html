Detailed Description

Caches InputControlLayout instances.

In the editor we need access to the InputControlLayouts registered with the system in order to facilitate various UI features. Instead of constructing layout instances over and over, we keep them around in here.

This class is only available in the editor (when UNITY_EDITOR is true).
